## 6. 风险与遗留问题

| 风险 / 遗留 | 影响 | 处置 |
|---|---|---|
| **`thinking_delta` 真实形态** | 代码按 Anthropic SDK 文档实现，但 MiniMax 实际 delta 字段名/形状未验证 | L1 测试用 mock 守住格式；Live 验证时真实响应无 thinking 块（本次 6 轮全无），说明 MiniMax-M3 默认不返回 extended thinking；如未来需要，调 `LLM_TEMPERATURE` / 加 `thinking` 参数即可 |
| **pnpm 11 strict scripts** | 每个 `pnpm` 命令都跑 deps 状态检查，缺 allow-builds 会卡 | 已在 `.npmrc` 写 `allow-builds[]=@biomejs/biome`、`allow-builds[]=esbuild` |
| **L3 测试用 `setTimeout` 间隔** | 行间 600ms 是凭经验定值，CI 慢机器可能 flaky | 后续如 flaky 改为"等待 stdout 含 `[Teacher]: ` 后再写下一行" |
| **Readline 关闭时的 exit code** | 早期 stdin 关闭会让 CLI exit 1；已捕获 `ERR_USE_AFTER_CLOSE` 并 break | L3 现在 exit 0 |
| **Mock SDK 的方式** | 用 `vi.mock` 整个模块 + 替换 `mockImplementation`；粗糙但够用 | v0.5 引入 tool calls 后需要更细的 mock，届时再改 |

## 7. 手动 smoke 验证（已做）

### 7.1 Live 模式（DoD #5 补做，2026-06-04）

真实 `MINIMAX_API_KEY` 填进 `.env` 后，用 `scripts/live-smoke.mjs` 喂 6 轮对话：

```
$ RUN_LIVE_LLM=1 node dist/cli.js
English Oral Teacher Agent — CLI mode
Persona target: Sample Student (age 13, intermediate)
Type "exit" to quit.

> [Teacher]: Hi there! I'm Alex. How are you today? 😊

> [Teacher]: I'm glad to hear that! What did you do today?

> [Teacher]: Oh, Minecraft is so cool! Did you build something or play survival mode?

> [Teacher]: That sounds awesome! What's the coolest castle you've ever built?

> [Teacher]: Oh no, creepers are the worst! Have you tried building a wall around your castle to keep them out?

> [Teacher]: Haha, smart move! Do you play with friends, or mostly by yourself?
```

6 轮对话全部成功，persona 一致：自报名字、emoji 装饰、跟学生兴趣（Minecraft / 城堡 / creepers）互动、每轮一个引导式提问。SOUL.md 的人设规则在 LLM 真实响应下被遵守。

### 7.2 Replay 模式（已做，2026-06-03）

3 轮 Replay smoke 同样跑通（见 cbf74ae commit 时验证）。

```bash
printf "hi\nfine thanks\nplayed minecraft\nexit\n" | MINIMAX_API_KEY=sk-test node dist/cli.js
```

输出含 3 次 `[Teacher]:` 响应，与 fixture 内容一致。
