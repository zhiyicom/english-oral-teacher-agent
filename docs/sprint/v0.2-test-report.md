## 6. 风险与遗留问题

| 风险 / 遗留 | 影响 | 处置 |
|---|---|---|
| **Live 验证未跑** | DoD #5 未完成 | 需要真实 `MINIMAX_API_KEY`；已在 `.env.example` 留位；sprint 收尾后任何时候可补 |
| **`thinking_delta` 真实形态** | 代码按 Anthropic SDK 文档实现，但 MiniMax 实际 delta 字段名/形状未验证 | L1 测试用 mock 守住格式；Live 验证时会暴露真实差异；不匹配时改 AnthropicProvider 即可 |
| **pnpm 11 strict scripts** | 每个 `pnpm` 命令都跑 deps 状态检查，缺 allow-builds 会卡 | 已在 `.npmrc` 写 `allow-builds[]=@biomejs/biome`、`allow-builds[]=esbuild` |
| **L3 测试用 `setTimeout` 间隔** | 行间 600ms 是凭经验定值，CI 慢机器可能 flaky | 后续如 flaky 改为"等待 stdout 含 `[Teacher]: ` 后再写下一行" |
| **Readline 关闭时的 exit code** | 早期 stdin 关闭会让 CLI exit 1；已捕获 `ERR_USE_AFTER_CLOSE` 并 break | L3 现在 exit 0 |
| **Mock SDK 的方式** | 用 `vi.mock` 整个模块 + 替换 `mockImplementation`；粗糙但够用 | v0.5 引入 tool calls 后需要更细的 mock，届时再改 |
