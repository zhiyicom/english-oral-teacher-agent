## 3. Issues

### 3.1 proper-lockfile requires file to exist before locking

`lockfile.lock(path)` 在文件不存在时抛出 ENOENT（lstat 失败）。解决方案：lock 前先检查文件是否存在，若不存在则从 `USER.md.example` 复制创建。

### 3.2 proper-lockfile 无 TypeScript 类型

创建了 `src/types/proper-lockfile.d.ts` 声明模块类型。
