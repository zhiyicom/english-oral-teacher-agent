## 1. 真正逐字流式对话

### 1.1 当前状态

```
LLM SDK stream → chatStreamWithRetry() → 缓冲拼接 → { response, usage }
→ runTurn() → 取 response → yield student-text（一次性全文）
→ SSE → 浏览器 → 整条消息渲染
```

### 1.2 目标状态

```
LLM SDK stream → chatStreamWithRetryGen() → yield ChatChunk（不缓冲）
→ runTurn() → yield text-chunk（逐 delta）→ ... → yield student-text（全文，保留兼容）
→ SSE → 浏览器 → 逐 delta 追加到气泡
```

### 1.3 实现方案

**新增 `chatStreamWithRetryGen` async generator**（`src/llm/retry.ts`）：

```typescript
export async function* chatStreamWithRetryGen(
  client: LLMClient,
  opts: ChatOpts,
  maxAttempts = 2,
): AsyncGenerator<ChatChunk, void, void> {
  let lastErr: unknown = null
  for (let attempt = 1; attempt <= maxAttempts; attempt++) {
    try {
      for await (const chunk of client.chatStream(opts)) {
        yield chunk
      }
      return
    } catch (err) {
      lastErr = err
      const classified = classifyLLMError(err)
      process.stderr.write(
        `[cli] llm error: ${classified.classification} (${(err as Error).message})\n`,
      )
      if (!classified.retryable) break
      if (attempt < maxAttempts) {
        process.stderr.write(`[cli] retrying in 1s... (attempt ${attempt + 1}/${maxAttempts})\n`)
        await new Promise((r) => setTimeout(r, 1000))
      }
    }
  }
  throw lastErr
}
```

**保持 `chatStreamWithRetry` 兼容**（用作 wrapper）：

```typescript
export async function chatStreamWithRetry(...): Promise<StreamResult> {
  let response = ''
  let usage = null
  for await (const chunk of chatStreamWithRetryGen(client, opts, maxAttempts)) {
    if (chunk.type === 'text') response += chunk.delta
    else if (chunk.type === 'usage') usage = chunk
  }
  return { response, usage }
}
```

**修改 `runTurn`**（`src/agent/turn.ts`）：
- 新增 `text-chunk` 到 `TurnEvent` union
- 将 `chatStreamWithRetry` 调用替换为 `chatStreamWithRetryGen`
- 循环中 yield `{ type: 'text-chunk', delta }` 给每个 text chunk
- 收集 usage chunk
- 循环后 yield `student-text`（全文，保持向后兼容）

**修改 CLI `forwardEvent`**（`src/cli.ts`）：
- 新增 `text-chunk` case：`process.stdout.write(event.delta)`

**修改 SessionPage**（`web/src/components/SessionPage.tsx`）：
- 监听 `text-chunk` SSE event
- 实时追加 delta 到当前 assistant 气泡（不新建，修改末尾）

### 1.4 向后兼容

- `student-text` 事件保持不变——旧版客户端忽略 `text-chunk` 仍能正常工作
- `chatStreamWithRetry` 函数签名不变——被 `runTurn` 保留为可用路径
