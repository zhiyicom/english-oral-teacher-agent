## 1. 技术方案

### 1.1 全浏览器端实现

不使用任何服务端或第三方 STT/TTS API。浏览器原生 API：
- `SpeechRecognition`（Chrome 内置，免费，无需 API key）
- `SpeechSynthesis`（所有现代浏览器内置）

零新 npm 依赖。

### 1.2 浏览器兼容

| API | Chrome | Edge | Firefox | Safari |
|---|---|---|---|---|
| SpeechRecognition | 全支持 | 全支持 | 不支持 | 部分支持 |
| SpeechSynthesis | 全支持 | 全支持 | 全支持 | 全支持 |

不支持的浏览器自动隐藏语音按钮（特性检测）。

## 2. STT 设计

### 2.1 组件结构

```
VoiceInput
├── [🎤] 麦克风按钮（idle 状态，灰色）
├── [🎤] 麦克风按钮（listening 状态，红色脉冲动画）
└── [🚫] 或隐藏（不支持时）
```

### 2.2 状态机

```
idle → (点击) → listening → (说话) → interim 更新输入框
                            → (点击或静默) → final 填入输入框 → idle
                            → (error) → idle
```

### 2.3 实现要点

```typescript
const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition

const recognition = new SpeechRecognition()
recognition.continuous = true    // 持续录音直到手动停止
recognition.interimResults = true // 实时显示中间结果
recognition.lang = voiceAccent   // 'en-US' 或 'en-GB'

recognition.onresult = (event) => {
  for (const result of event.results) {
    if (result.isFinal) {
      // 填入输入框
      onFinal(result[0].transcript)
    } else {
      // 实时显示在输入框
      onInterim(result[0].transcript)
    }
  }
}
```

## 3. TTS 设计

### 3.1 触发时机

LLM 回复到达时（`text-chunk` 事件），如果 `voice_enabled = true`：
1. 等待第一个 text-chunk 到达
2. 累积完整文本
3. `done` 事件后，用 `SpeechSynthesis` 朗读全文

不流式朗读（边生成边读有技术复杂度，v0.9.1 考虑）。

### 3.2 实现要点

```typescript
function speak(text: string, rate: number, accent: string) {
  const utterance = new SpeechSynthesisUtterance(text)
  utterance.rate = rate       // 0.5 - 2.0
  utterance.lang = accent     // 'en-US' / 'en-GB'
  utterance.pitch = 1.0

  // 选择对应口音的语音
  const voices = speechSynthesis.getVoices()
  const voice = voices.find(v => v.lang.startsWith(accent))
  if (voice) utterance.voice = voice

  speechSynthesis.speak(utterance)
}
```

### 3.3 中断

- 新 turn 开始时取消当前朗读（`speechSynthesis.cancel()`）
- 用户点击发送时取消朗读

## 4. Settings 集成

### 4.1 控件启用

三个语音控件从 `disabled` 变为可用：

| 控件 | 类型 | 存储 | 默认值 |
|---|---|---|---|
| voice_enabled | toggle | USER.md | false |
| voice_speed | slider (0.5–2.0) | USER.md | 1.0 |
| voice_accent | select (en-US/en-GB) | USER.md | en-US |

### 4.2 数据流

```
Settings 页面修改 → PUT /api/settings → USER.md（持久化）
                 → localStorage（即时生效）
                 → SessionPage 读取 localStorage
```
