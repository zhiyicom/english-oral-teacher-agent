# Sprint v0.9 范围 — 语音输入输出 (STT + TTS)

> **Sprint**: v0.9（F6 语音 I/O）
> **父文档**: PRD §6.6 语音功能
> **上一 sprint**: v0.8.5（v0.8 Web UI 完整交付 + sidebar 重构）
> **分支**: `main`
> **日期**: 2026-06-14

---

## 背景

v0.8 全部 5 个 sub-sprint 已完成，Web UI 拥有完整会话功能 + sidebar 布局。v0.9 按 PRD F6 计划实现语音输入输出。

语音功能使用浏览器原生 API（零额外成本，零新依赖）：
- **STT**: `SpeechRecognition` API（Chrome 内置）
- **TTS**: `SpeechSynthesis` API（浏览器内置语音引擎）

## 2. 范围外

| 事项 | 原因 |
|---|---|
| 服务端 TTS/STT | 浏览器原生 API 已足够 |
| 流式 TTS（边生成边读）| 需要更复杂的缓冲策略，v0.9.1 |
| 移动端语音兼容 | iOS Safari 的 Speech API 有限制，暂不处理 |
| 语音唤醒词 | 未在 PRD 中 |
| 多语言 TTS 切换 | 中文 TTS 引擎与英文不同，暂不混用 |
