# Sprint v1.0.1 范围 — 功能优化打磨

> **Sprint**: v1.0.1（v0.9 语音功能之外的优化打磨）
> **上一 sprint**: v0.9（F6 语音 I/O）
> **分支**: `main`
> **日期**: 2026-06-15

---

## 背景

v0.9 实现了语音输入输出。在测试过程中发现多项需要打磨的问题：Phase 推进不够强制、话题追踪不准确、会话结束流程不完整、LLM 约束力不够强。本 sprint 集中修复这些问题。

---
